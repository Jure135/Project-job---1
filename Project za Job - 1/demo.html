Main Website

Complete homepage with booking interface, location picker, vehicle selection, and real-time pricing. Includes user registration, payment integration, and trip history.

Admin Dashboard

Comprehensive admin panel for managing users, drivers, bookings, pricing, promotions, and customer support. Includes analytics and live trip tracking.

View Admin Panel

Advanced Booking System

Supports one-way, return, airport, and port transfers. Multiple vehicle types, passenger/luggage selection, and real-time price calculation with surge pricing.

Live Trip Tracking

Real-time GPS tracking of active trips with Google Maps integration. Both customers and admins can track driver location and trip progress.

Payment Integration

Multiple payment methods including credit/debit cards, PayPal, Apple Pay, and Google Pay. Secure processing with transaction history.

User & Driver Management

Complete user registration system with profile management. Driver verification, document management, and status control from admin panel.

Promotional System

Advanced promo code system with percentage and fixed discounts. Bulk creation, usage tracking, and expiry management.

Customer Support

Complete support system with live chat, ticket management, FAQ section, and complaint handling. Admin can manage all support requests.

Analytics & Reports

Comprehensive analytics dashboard with revenue tracking, trip statistics, user metrics, and performance indicators. Export functionality included.
